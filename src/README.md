# Source layout

This repo holds **HLD**, **LLD**, **design patterns**, **SOLID principles**, and **basics** (OOP, etc.) for practice and reference.

| Folder | Purpose |
|--------|--------|
| [**HLD**](HLD/) | High Level Design – practice problems and solutions |
| [**LLD**](LLD/) | Low Level Design – practice questions and solutions |
| [**DesignPatterns**](DesignPatterns/) | Creational, behavioural, and structural design patterns with Before/After examples |
| [**Principles**](Principles/) | **SOLID** and other design principles – see [Principles/SOLID/README.md](Principles/SOLID/README.md) (BadCode vs GoodCode) |
| [**Basics**](Basics/) | Fundamentals: OOP concepts, access modifiers, inheritance, etc. |

Run examples from each subfolder (e.g. `DesignPatterns/CreationalPatterns/...`, `Principles/SOLID/...`) using their own `Main` classes.
